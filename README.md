# 🏥 Medical Data History (SQL)

## 📌 Project Overview

Medical Data History is a SQL-based analytics project focused on querying healthcare datasets to answer real-world business questions. The project demonstrates proficiency in SQL by solving multiple analytical queries using joins, aggregate functions, grouping, subqueries, and filtering techniques.
